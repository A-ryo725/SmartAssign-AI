# SmartAssign-AI
ハッカソン提出用：生成AIとシステムを融合させたハイブリッド型タスク管理ツール
生成AIの「柔軟性」とシステムの「確実性」を融合させた、実用性と堅牢性を極限まで高めたタスク管理ツールです。

## 作品概要・デモ動画
詳細なアーキテクチャの解説、工夫したポイント、および動作デモ動画は **Zennの記事** にすべてまとめています。
**https://zenn.dev/kit_product/articles/39819340d6b887**

## 技術スタック & 構成
- **フロントエンド / エージェント基盤:** Copilot Studio（Web検索OFFによるハルシネーション対策）
- **バックエンド / ロジック実行:** Power Automate
  - フローの実装（型エラー・上書き・ゴミデータ対策の例外処理を内包）
- **データベース:** SharePoint リスト

## 提出ファイルについて
本リポジトリに含まれる `.zip` ファイルは、Copilot StudioのエージェントおよびPower Automateのフロー一式を含む「アンマネージドソリューション」です。
